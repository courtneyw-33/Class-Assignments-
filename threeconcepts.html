3 Programming Concepts

String Manipulation

String manipulation is the set of techniques programmers use to modify, analyze, or extract information from text data (strings).
String manipulation means applying operations like changing case, measuring length, slicing, searching, splitting, replacing, and formatting to transform or analyze text.

Functions

A function in programming is a named block of code that performs a specific task and can be reused whenever needed.
Functions may take inputs (parameters), perform an operation, and optionally return a result. It helps make programs modular, organized, and easier to maintain.

Loops

A loop in programming is a control structure that lets you repeat a block of code multiple times until a certain condition changes.
The repeated instructions, as long as a condition is true, makes it possible to handle repetitive tasks efficiently.
